# Overview

A web app where people create and manage their to-do lists. Each person sees only their own tasks.

# Who uses it

- Individual users — they track what they need to get done.

# What they can do

- Sign up and log in to see their tasks.
- Add a new task with a title and optional due date.
- Mark a task as complete or incomplete.
- Organize tasks into categories they create themselves.
- Edit or delete tasks they've added.
- Monitoring based on last month todos
